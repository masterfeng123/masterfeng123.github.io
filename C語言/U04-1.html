U04-1 數字平方和

題目敘述：

和 n）之間所有整數的平方和。

輸入範例
2 4

輸出範例
29


#include <stdio.h>

int main()
{
	int m, n, sum, i, temp;
	scanf("%d %d", &m, &n);
	if (n>m){
		temp=m;
		m=n;
		n=temp;
	}
	for(i=n;i<=m;i++){
		sum+=i*i;
	}
	printf("%d", sum);
	return 0;
}

先輸入兩數至n,m，然後確定n小於m，接著利用迴圈取n~m，每一次都將平方加進sum，最後sum就是計算結果。
