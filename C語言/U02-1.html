U02-1大寫換小寫

題目敘述：

輸入三個大寫字元，將它們變成小寫字元後輸出。

輸入範例
BIG

輸出範例
big


#include <stdio.h>

int main()
{
	char a, b, c;
	int d= 32;
	scanf("%c%c%c", &a, &b, &c);
	printf("%c%c%c", a+d, b+d, c+d);
	return 0;
}

設定三個字元a,b,c，接著宣告d=32(因為大小寫位元相差32)，將三個字元輸入後加上位元差，使其印出時為小寫字元
