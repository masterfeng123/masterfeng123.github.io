U12-2 鍵盤移位

題目敘述：

把手放在鍵盤上時，稍不注意就會往右錯一位。這樣，輸入Q會變成輸入W，輸入J會變成輸入K等。
輸入一個錯位後敲出的字串(所有字母均大寫)，輸出打字員本來想打出的句子。輸入保證合法，即一定是錯位之後的字串。例如輸入中不會出現大寫字母A。

輸入範例
O S, GOMR YPFSU/

輸出範例
I AM FINE TODAY.


#include <stdio.h>

int main()
{
	char keyboard[]="`1234567890-=QWERTYUIOP[]\ASDFGHJKL;'ZXCVBNM, ./";
	char a[1000];
	gets(a);
	for(int i=0;a[i];i++){
		if(a[i]==' ') printf(" ");
		else{
			for(int j=0;keyboard[j];j++){
				if(a[i]==keyboard[j]){
					printf("%c", keyboard[j-1]);
					break;
				}
			}
		}
	}
	return 0;
}
