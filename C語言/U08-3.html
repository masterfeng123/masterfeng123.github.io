U08-3數列兩兩對調

題目敘述：

先輸入5個整數X1~X5。接著再每次輸入兩個 1 到 5 的數 m, n，然後將Xm和Xn的位置對調，重複輸入直到檔尾為止。最後印出這 5 個數。 (注意：輸出的開頭有一個空白。)

範例輸入
75 40 44 6 73
5 1
5 1
1 2
5 5

範例輸出
40 75 44 6 73


#include <stdio.h>

int main()
{
	int a[5];
	for(int i=0;i<5;i++){
		scanf("%d", &a[i]);
	}
	int b, c, temp;
	while(scanf("%d %d", &b, &c)!=EOF){
		temp=a[c-1];
		a[c-1]=a[b-1];
		a[b-1]=temp;
	}
	for(int i=0;i<5;i++){
		printf(" %d", a[i]);
	}
	return 0;
}

將第一行輸入a陣列。接著迴圈持續運作到檔尾，每一次都將對應索引值內容互換，最後印出。
