U10-3 爬樓梯

題目敘述：

小明爬樓梯，已知要爬的梯數有 N (N<20) 階，但小明一次可以爬 1~3 階，請問總共有幾種爬法？（提示：使用遞迴，找 F(n) 的遞迴式）

輸入範例
3

輸出範例
4


#include <stdio.h>

int main()
{
	int n;
	scanf("%d", &n);
	printf("%d", ccc(n));
	return 0;
}

int ccc(int n){
	if(n==1) return 1;
	if(n==2) return 2;
	if(n==3) return 4;
	if(n>3) return ccc(n-3)+ccc(n-2)+ccc(n-1);
}

ccc()函式中，剩一階回傳1，剩二階回傳2，剩三階回傳4，剩三階以上，則回傳剩三階、剩兩階、剩一階的方法數。
