U03-3 判斷及調整成績

題目敘述：

輸入成績 score，
假設大於等於 90 分以上，評定等級為 A，且 score 改成 95；
假設介於 [80, 90) 之間，評定等級為 B，且 score 改成 85；
假設介於 [70, 80) 之間，評定等級為 C，且 score 改成 75；
假設介於 [60, 70) 之間，評定等級為 D，且 score 改成 65；
最後如果低於 60，評定等級為 F，且 score 不變。
最後請將等級及成績做輸出。

輸入範例
68

輸出範例
D 65


#include <stdio.h>

int main()
{
	int a;
	scanf("%d", &a);
	if(a>=60){
		if(a>=70){
			if(a>=80){
				if(a>=90){
					a=95;
					printf("A %d", a);
				}
				else
				{
					a=85;
					printf("B %d", a);
				}
			}
			else{
				a=75;
				printf("C %d", a);
			}
		}
		else
		{
			a=65;
			printf("D %d", a);
		}
	}
	else{
		printf("F %d", a);
	}
	return 0;
}

利用巢狀式判斷句判斷實際分數落在哪一個區間，並給予相應調整。
