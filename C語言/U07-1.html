U07-1 倒印數列

題目敘述：

輸入n以及n個整數(n<=30)，將此n個整數倒印出來

輸入範例
5 4 3 2 1 0

輸出範例
0 1 2 3 4


#include <stdio.h>

int main()
{
	int n;
	scanf("%d", &n);
	int a[n];
	for(int i=0;i<n;i++){
		scanf("%d", &a[i]);
	}
	for(int i=0;i<n;i++){
		printf("%d", a[n-i-1]);
		if(i!=n-1) printf(" ");
	}
	return 0;
}

先輸入n，然後宣告n個空間的陣列a，再使用迴圈輸入數列，再用迴圈倒過來印。
