U05-2 求兩數和(非零)

題目敘述：

重覆讀取兩個數，並印出該兩數的和，直到兩個數皆為 0 為止。

輸入範例
574 949
113 854
702 229
0 0

輸出範例
1523
967
931


#include <stdio.h>

int main()
{
	int a=3, b=3;
	while(a!=0 && b!=0){
		
		scanf("%d %d", &a, &b);
		if(a==0 && b==0){
			break;
		}
		printf("%d\n", a+b);
	}
	return 0;
}

設while迴圈，並設定條件讓a=0且b=0時跳出迴圈。
