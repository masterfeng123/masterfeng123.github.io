U11-2 反數相加

題目敘述：

輸入一個整數，將其前後顛倒後，得到一個新的數，求原數與新數之和。例如輸入 124，顛倒後得到 421，相加為 545。

輸入範例 #1
124

輸出範例 #1
545

輸入範例 #2
258

輸出範例 #2
1110


#include <stdio.h>

int main()
{
	int n, c=0, b;
	scanf("%d", &n);
	b=n;
	while(b>0){
		c=c*10+b%10;
		b/=10;
	}
	printf("%d", c+n);
	return 0;
}
