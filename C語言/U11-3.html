U11-3 第k個質數

題目敘述：

第 1 個質數是 2，第 2 個質數是 3，第 3 個質數是 5，…，現在輸入 k，程式須輸出第 k 個質數。(k 小於1000)

輸入範例
1

輸出範例
2


#include <stdio.h>

int main()
{
	int k, check;
	scanf("%d", &k);
	for(int i=1;check<=k;i++){
		if(ccc(i)==1){
			check++;
			if(check==k){
				printf("%d", i);
				break;
			}
		}
	}
	return 0;
}

int ccc(int n){
	int f=0;
	for(int i=1;i<=n;i++){
		if(n%i==0) f++;
	}
	if(f==2) return 1;
	return 0;
}
