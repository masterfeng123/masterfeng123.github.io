U11-1 開燈問題

題目敘述：

有 n盞燈，編號為 1~n。第 1 個人把所有燈打開，第 2 個人按下所有編號為 2 的倍數的開關 (這些燈將被關掉)，第 3 個人按下所有編號為 3 的倍數的開關 (其中關掉的燈將被打開，開著的燈將被關閉)，依此類推。
一共有 k 個人，問最後有哪些燈開著?

Input
n 和 k

Output
最後開著的燈的編號

Limit
1≤k≤n≤1000

Sample Input
7 3

Sample Output
1 5 6 7


#include <stdio.h>

int main()
{
	int n, k, m=0;
	scanf("%d %d", &n, &k);
	int light[1005]={0};
	for(int i=1;i<=k;i++){
		for(int j=0;j<n;j++){
			if((j+1)%i==0){
				if(light[j]==0) light[j]=1;
				else if(light[j]==1) light[j]=0;
			}
		}
	}
	for(int i=0;i<n;i++){
		
		if(light[i]==1){
			if(m==1) printf(" ");
			printf("%d", i+1);
			m=1;
		}
	}
	return 0;
}
