U04-3 質數判別

題目敘述：

輸入一個正整數，如果是質數，則輸出 Yes，如果不是，則輸出 No。

輸入範例
3

輸出範例
Yes


#include <stdio.h>

int main()
{
	int n, i, ch;
	ch=0;
	scanf("%d", &n);
	for(i=2;i<n;i++){
		if(n%i==0){
			printf("No");
			ch=1;
			break;
		}
	}
	if (ch!=1){
		printf("Yes");
	}
	return 0;
}

設未知數n，使用迴圈讓因數從2找到n(這裡不包含n，n必為自己的因數，所以我忽略了)，若在2~n-1當中找到數字可以整除，則判斷其不為
質數，跳出迴圈。
