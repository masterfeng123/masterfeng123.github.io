U09-4 字串串接

題目敘述：

本題主要是串接不同的密碼字串，所有字串都由數字和英文字母構成。第一行是一個字串，接下來每一行會出現一個數字及一個字串，如果數字是 0，則後面出現的字串要串接在前面；如果數字是 1，則後面出現的字串要串接在後面，如以下範例所示。輸入行數不定，讀至檔尾結束。程式須印出最後串接的字串，總長不會超過 150 個字元。

輸入範例
abc
1 def
0 123

輸出範例
123abcdef


#include <stdio.h>

int main()
{
	char a[150], b[150];
	gets(a);
	int n;
	while(scanf("%d %s", &n, b)!=EOF){
		if(n==1){
			strcat(a, b);
		}
		else{
			strcat(b, a);
			for(int i=0;i<150;i++){
				a[i]=b[i];
			}
		}
	}
	printf("%s", a);
	return 0;
}

strcat(a,b)的功能是將字串陣列接在a後面。首先輸入a字串，然後在依照0與1的規則將字串接在後面，而因為目標字串是a，因此a接在b後面時，要再將a轉換成b字串。
