U10-2費式數列

題目敘述：

已知 f(0)=0, f(1)=1, 且 f(n)=f(n-1)+f(n-2), n>=2。 使用遞迴求 f(n) = ?

輸入範例
5

輸出範例
f(5) = 5


#include <stdio.h>

int main()
{
	int n;
	scanf("%d", &n);
	printf("f(%d) = %d", n, ccc(n));
	return 0;
}

int ccc(int n){
	if(n==0) return 0;
	else if(n==1) return 1;
	else return ccc(n-1)+ccc(n-2);
}

此處設置ccc()函式，功能式依照費式數列規律呼叫下一個ccc()函式。直接將輸入的n導入ccc()函式即可。
