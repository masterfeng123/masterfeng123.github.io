U08-1印出字串及其長度

題目敘述：

輸入一字串，將此字串及其長度印出。（字串不包含空白）

輸入範例
example

輸出範例
STR = example, LEN = 7


#include <stdio.h>

int main()
{
	int n;
	char b[1000];
	gets(b);
	for(int i=1;b[i-1]!='\0';i++){
		n=i;
	}
	printf("STR = %s, LEN = %d", b, n);
	return 0;
}

設定1000個空間的字串陣列，使用gets()函式輸入b，設定迴圈重複直到b陣列的該索引值內容為空。
