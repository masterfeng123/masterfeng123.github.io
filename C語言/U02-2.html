U02-2 圓柱表面積

題目敘述：

輸入圓柱的半徑與高度(實數)，輸出其表面積至小數點以下兩位(四捨五入)。(π=3.14159)

輸入範例
10.4 2.3

輸出範例
829.88


#include <stdio.h>

int main()
{
	float a, b;
	scanf("%f %f", &a, &b);
	printf("%.2f", (a*a*3.14159*2)+(a*2*3.14159*b));
	return 0;
}

設定浮點數a,b(這樣才能除出小數點)，掃描高度與半徑，直接在prtinf函式中輸入計算式。小數點取兩位使用.2f，若三位使用.3f以此類推。
