U05-3 求兩數和(檔尾)

題目敘述：

任意給兩個數，印出兩數之和。程式須能反覆讀取數對（每次兩個），並印出結果，直到讀不到數字為止（檔尾結束）。

輸入範例
574 949
113 854
702 229
351 239
525 52
170 622

輸出範例
1523
967
931
590
577
792


#include <stdio.h>

int main()
{
	int a, b;
	while(scanf("%d %d", &a, &b) != EOF) {

		printf("%d\n", a+b);
	}
	return 0;
}

while(scanf("%d %d", &a, &b)!=EOF)的意思是當scanf還偵測得到數值時迴圈繼續，偵測不到數值則跳出迴圈。
