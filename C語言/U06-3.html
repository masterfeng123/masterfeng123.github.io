U06-3 求兩個數的最大公因數

題目敘述：

輸入兩個整數，由程式印出其最大公因數。

輸入範例
18 126

輸出範例
The GCD of 18 and 126 = 18 


#include <stdio.h>

int main()
{
	int a, b, i, gcd;
	scanf("%d %d", &a, &b);
	for(i=1;i<=a;i++){
		if(a%i==0 && b%i==0){
			gcd=i;
		}
	}
	printf("The GCD of %d and %d = %d", a, b, gcd);
	return 0;
}

設1~a的迴圈，尋找同時能被a,b整除的數，因為是從小比到大，故gcd會獲得最大
公因數。
