U10-4 河內塔問題

題目敘述：

給定圓盤個數 n，依河內塔規則，以最快步驟從 A 柱移到 C 柱 (B 柱為緩衝) ，試印出所有移動步驟，格式請參見範例。

輸入範例
2

輸出範例
Move D1 from A to B
Move D2 from A to C
Move D1 from B to C


#include <stdio.h>

int main()
{
	int n;
	scanf("%d", &n);
	ccc(n, 'A', 'C', 'B');
	return 0;
}


void ccc(int n, char start, char tar, char middle){
	if(n>0){
		ccc(n-1, start, middle, tar);
		printf("Move D%d from %c to %c\n", n, start, tar);
		ccc(n-1, middle, tar, start);
	}
}

依據河內塔規則，先換到中間再換到最旁邊的方法，寫出ccc()函式。
