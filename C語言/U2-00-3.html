U2-00-3 求位數和

題目敘述：

任意輸入一個正整數，求該整數的每個數字和。程式需重覆執行，直到檔尾結束。(舊題)

輸入範例
123
45

輸出範例
6
9


#include <stdio.h>

int main()
{
    int a, tatal=0, b;
    while(scanf("%d", &a) != EOF){
        tatal=0;
        while(a>=10){
            
            tatal+=a%10;
            
            a/=10;
            
        }
        if (a/10==0){
            tatal+=a/10;
        }
        tatal+=a%10;
        printf("%d\n", tatal);
    }
    return 0;
}
