U2-00-4 倒三角形

題目敘述：

利用n在迴圈中數值漸漸變小的特性，每跑一次迴圈，n就-1，使*能印出倒三角形的形狀。(舊)


#include <stdio.h>

int main()
{
    int n, i;
    scanf("%d", &n);
    while(n>0){
        for(i=0;i<n;i++){
            printf("*");
        }
        printf("\n");
        n-=1;
    }
    return 0;
}
