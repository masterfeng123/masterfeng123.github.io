U09-2印三角形函數

題目敘述：

寫一個程式，能根據輸入行數及符號列印三角形形狀，如以下範例所示。

輸入範例 #1
3#

輸出範例 #1
#
##
###

輸入範例 #2
5@

輸出範例 #2
@
@@
@@@
@@@@
@@@@@


#include <stdio.h>

int main()
{
	int n;
	char a;
	scanf("%d%c", &n, &a);
	for(int i=1;i<=n;i++){
		for(int j=0;j<i;j++){
			printf("%c", a);
		}
		printf("\n");
	}
	return 0;
}

偵測目標字元，然後使用雙層迴圈方式印出字元。
