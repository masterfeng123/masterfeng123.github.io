U04-2 求數字和

題目敘述：

輸入 n，計算從 1 到 n 當中，不是 2 或 3 或 5 的倍數的數字和。

輸入範例
10

輸出範例
8


#include <stdio.h>

int main()
{
	int n, i, sum, ch;
	scanf("%d", &n);
	for(i=1;i<=n;i++){
		if(i%2==0){
			continue;
		}
		if(i%3==0){
			continue;
		}
		if(i%5==0){
			continue;
		}
		sum+=i;
		
	}
	printf("%d", sum);
	return 0;
}

設未知數n，從最小1~n當中，i整除2、3或5的時候跳過這個迴圈進到下一個迴圈。
