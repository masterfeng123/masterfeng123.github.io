U11-4得分

題目敘述：

輸入一個 O 和 X 組成的字串 (長度為 1~50)，輸出其得分。計分規則如下：每個 O 的得分為目前連續出現的 O 的個數，X 的得分為 0。例如：OOXXOXXOOO 的得分為 1+2+0+0+1+0+0+1+2+3=10。

輸入範例
OOXXOXXOOO

輸出範例
10


#include <stdio.h>

int main()
{
	int c=1, sum=0;
	char a[50];
	gets(a);
	for(int i=0;a[i];i++){
		if(a[i]=='O'){
			sum+=c;
			c++;
		}
		else{
			c=1;
		}
	}
	printf("%d", sum);
	return 0;
}
