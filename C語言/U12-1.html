U12-1 生成元

題目敘述：

如果一個數 x 加上 x 的各個數字之和得到 y，則稱 x 是 y 的一個生成元，例如 x=198，則 y=198+1+9+8=216，因此 198 是 216 的生成元。另一方面，我們也可以發現當 x=207 時，y=207+2+0+7=216，因此 207 也是 216 的生成元。本題要求輸入整數 n，程式須列出 n 的最小生成元。如果找不到生成元，則印出 no generator。輸入部份每行一個數，讀到檔尾結束；輸出的部份則針對每個輸入，輸出相對應的結果。

輸入範例
216
20

輸出範例
198
no generator


#include <stdio.h>

int main()
{
	int y, check=0;
	while(scanf("%d", &y)!=EOF){
		for(int i=1;i<=y;i++){
			check=i;
			check+=ccc(i);
			if(check==y){
				printf("%d\n", i);
				break;
			}
			if(i==y){
				printf("no generator\n");
				break;
			}
		}
	}
	return 0;
}

int ccc(int n){
	int d=0;
	while(n>0){
		d+=n%10;
		n/=10;
	}
	return d;
}
