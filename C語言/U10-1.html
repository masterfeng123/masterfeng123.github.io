U10-1 3n+1問題

題目敘述：

考慮以下的演算法：
1.輸入 n
2.印出 n
3.如果 n = 1 結束
4.如果 n 是奇數，則令 n=3*n+1；否則令 n=n/2
5.GOTO 2
例如輸入 22，則得到數列 22 11 34 17 52 26 13 40 20 10 5 16 8 4 2 1。此數列長度為 16，我們稱 16 為 22 的循環長度。
本問題是對任2個整數i,j，想要知道介於這兩數之間所有的數的循環長度的最大值。

輸入格式
可能包含任意列測試資料，每一列有一對正整數i,j(0<i,j<1000000)

輸入範例
1 10
100 200
201 210
900 1000

輸出範例
1 10 20
100 200 125
201 210 89
900 1000 174 


#include <stdio.h>

int main()
{
	int i, j, a, b, lens;
	while(scanf("%d %d", &i, &j)!=EOF){
		int max=0;
		if(i>j){
			b=i;
			a=j;
		}
		else{
			a=i;
			b=j;
		}
		for(int c=a;c<=b;c++){
			lens=ccc(c);
			if(lens>max){
				max=lens;
			}
		}
		printf("%d %d %d\n", i, j, max);
	}
	return 0;
}

int ccc(int n){
	if(n==1) return 1;
	else if(n%2==0) return ccc(n/2)+1;
	else if(n%2!=0) return ccc(3*n+1)+1;
}

此處設置ccc()自訂函數，功能是依照題目演算法呼叫下一個ccc函式。因為不知道i,j哪一個數較大，因此先做比較，接著以此兩數作為迴圈參數，最後比較並提取出最大值
