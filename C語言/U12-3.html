U12-3 蛇行填數

題目敘述：

在 n×n 方陣裡填入1, 2, …, n×n (n≤8)，要求填成蛇形，從右上角出發，依順時針方向轉至中心點，如參考範例所示。 注意：使用 %3d 輸出整數格式。

輸入範例
4

輸出範例
10 11 12 1
9 16 13 2
8 15 14 3
7 6 5 4


#include <stdio.h>

int n, dir=2, a[9][9]={ {0} };
int row, column, num=0;

int main()
{
	scanf("%d", &n);
	row=1;
	column=n;
	a[row][column]=++num;
	while(1){
		switch(dir){
			case 0:while(row>1 && a[row-1][column]==0) {a[--row][column]=++num;} dir=1; break;
			case 1:while(column<n && a[row][column+1]==0) {a[row][++column]=++num;} dir=2; break;
			case 2:while(row<n && a[row+1][column]==0) {a[++row][column]=++num;} dir=3; break;
			case 3:while(column>1 && a[row][column-1]==0) {a[row][--column]=++num;} dir=0; break;
		}
		if(num>=n*n){
			break;
		}
	}
	for(int i=0;i<n;i++){
		for(int j=0;j<n;j++){
			printf("%3d", a[i+1][j+1]);
		}
		printf("\n");
	}
	return 0;
}

dir為填數方向，row、column為目前位置，因為n小於8，可以設一個9x9且內容為空的a矩陣。num為目前記數，while迴圈執行到num=nxn的時候跳出迴圈，即代表矩陣完成。
