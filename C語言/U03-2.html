U03-2 四數排序

題目敘述：

輸入四個正整數a、b、c、d，將a、b、c、d從小排到大並輸出。

輸入範例
2 8 5 3

輸出範例
2 3 5 8


#include <stdio.h>

int main()
{
	int a, b, c, d;
	scanf("%d %d %d %d", &a, &b, &c, &d);
	int x[]={a, b, c, d};
	int j, i, n;
	for(j=0;j<4;j++){
		for(i=0;i<3;i++)
			{
			int temp;
			if(x[i]>x[i+1])
			{
				temp=x[i+1];
				x[i+1]=x[i];
				x[i]=temp;
			}
		}
	}
	for(n=0;n<3;n++){
		printf("%d ", x[n]);
	}
	printf("%d", x[3]);
	return 0;
}

這裡使用簡單的氣泡排序法完成工作，這能使程式的複雜度大幅減少。
