U05-1 求兩數和(N次)

題目敘述：

程式先讀取 N，接下來重覆 N 次，每次任意給兩個數，印出兩數之和。

輸入範例
3
574 949
113 854
702 229

輸出範例
1523
967
931 


#include <stdio.h>

int main()
{
	int n, i, a, b;
	scanf("%d", &n);
	for(i=0;i<n;i++){
		scanf("%d %d", &a, &b);
		printf("%d\n", a+b);
	}
	return 0;
}

先讀取n，然後設次數n次的迴圈掃描每一行並印出來。
