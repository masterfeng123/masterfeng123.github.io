U08-2 字串大小寫個數

題目敘述：

任意輸入一字串，求出其中大寫及小寫英文字母各有幾個？

輸入範例
GoodJob

輸出範例
Up = 2, Low = 5


#include <stdio.h>

int main()
{
	int up=0, low=0;
	char b[1000];
	gets(b);
	for(int i=0;b[i]!='\0';i++){
		if(b[i]>='A' && b[i]<='Z'){
			up+=1;
		}
		if(b[i]>='a' && b[i]<='z'){
			low+=1;
		}
	}
	printf("Up = %d, Low = %d", up, low);
	return 0;
}

輸入字串進到陣列b，設定迴圈重複直到內容為空，每一次都會判斷該內容的儲存位元在大寫區間還是小寫區間。
