U02-3 整數除法

題目敘述：

由使用者輸入兩個整數，將其存到整數變數中，再計算它們的實數相除的商。例如輸入的值為 2 和 5，則輸出的結果應為 0.40 (用 %.2f 取兩
位小數)。

輸入範例
2 5

輸出範例
0.40


#include <stdio.h>

int main()
{
	float a, b;
	scanf("%f %f", &a, &b);
	printf("%.2f", a/b);
	return 0;
}

掃描a,b浮點數，輸出運算式結果。
