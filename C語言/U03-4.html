U03-4 判斷大小月

題目敘述：

輸入月份，如果是大月 (31日)，則印出 Yes；如果是小月，則印出 No；如果不是正確的月份，則印出 Error。

輸入範例
8

輸出範例
Yes


#include <stdio.h>

int main()
{
	int a;
	scanf("%d", &a);
	switch(a)
	{
		case 1:printf("Yes");break;
		case 2:printf("No");break;
		case 3:printf("Yes");break;
		case 4:printf("No");break;
		case 5:printf("Yes");break;
		case 6:printf("No");break;
		case 7:printf("Yes");break;
		case 8:printf("Yes");break;
		case 9:printf("No");break;
		case 10:printf("Yes");break;
		case 11:printf("No");break;
		case 12:printf("Yes");break;
		default:printf("Error");break;
	}
	return 0;
}

利用switch的語法寫出能判斷輸入的月份並判斷其是否為大小月。
