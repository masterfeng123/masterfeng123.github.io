U09-1 1+(1+2)+(1+2+3)...

題目敘述：

輸入一個正數 n(n≤100)計算 1+(1+2)+(1+2+3)+…+(1+2+…+n)。

輸入範例
5

輸出範例
35


#include <stdio.h>

int main()
{
	int n, sum=0;
	scanf("%d", &n);
	for(int i=1;i<=n;i++){
		for(int j=1;j<=i;j++){
			sum+=j;
		}
	}
	printf("%d", sum);
	return 0;
}

依照規律，每一次i都會+1，而第二層迴圈j再讓sum從1加到i。
